Some code licenses of dubious abilities:
Here's some code licenses I found that explicitly forbid using the code for training large language models. US courts often interpret copyright law such that using publicly accessible code for training is a form of fair use, so this is unlikely to do anything. But (not legal advice) it can't hurt.

genAI blocking Firefox extension:
Here's a rudimentary Firefox extension I made that blocks:
  1. Google's AI search (which you cannot disable, and thinks an honest question is the same as a statement of truth leading to disturbing outright lies)
  2. Quora's Poe bot
  3. ChatGPT - if you open it, you get met with a clown emoji
Of course this list is far from complete, but I still find it useful for myself.
